SIDEKICK

O Poder da Parceria Perfeita.

O que ela faz?

A Sidekick potencializa a mecânica de Tag-In. Ela é voltada para jogadores que trocam de personagem constantemente para manter o ritmo ou para salvar o parceiro de situações perigosas.

  • Entrada Fortalecida: Ataques realizados imediatamente após uma troca (Tag-in) ganham propriedades de prioridade ou dano aumentado.
  • Recuperação de Tag: Reduz levemente o tempo que o personagem que saiu da tela leva para estar disponível novamente para assistência.
  • Follow-up de Assistência: Facilita a conexão entre um ataque de assistência e a entrada definitiva do parceiro na luta.

Top 6 Times Sugeridos

Vi + Ekko

Troca dinâmica. A Vi entra quebrando a guarda e o Ekko utiliza a Fuse Sidekick para garantir que sua entrada seja rápida o suficiente para continuar o combo de rebobinar.

Yasuo + Ahri

Fluxo constante. Use o Yasuo para controlar o neutro e troque para a Ahri com Sidekick para garantir que o Charm dela pegue o inimigo desprevenido na entrada.

Darius + Warwick

Caçada brutal. Darius inicia o sangramento e Warwick entra com Sidekick para morder o alvo com ainda mais força, mantendo a pressão máxima.

Jinx + Caitlyn

Revezamento de tiro. Sidekick permite que a Caitlyn entre cobrindo a Jinx com precisão, garantindo que o time nunca fique vulnerável durante a troca de atiradoras.

Illaoi + Braum

Troca defensiva. Use a Sidekick para trazer o Braum para a tela com o escudo já ativo, protegendo a Illaoi enquanto ela se recupera para uma nova ofensiva.

Ahri + Ekko

Reset de Mix-up. Este time foca em confundir o oponente. Com Sidekick, as trocas se tornam ferramentas de ataque por si só, criando um jogo imprevisível.

Dica de Especialista

A Sidekick é a melhor Fuse para aprender o tempo de Handshake Tag. Pratique trocar no meio de ataques que normalmente seriam punidos; a Fuse garantirá que seu parceiro entre com vantagem de frames!
